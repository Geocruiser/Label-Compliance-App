# Wine TTB Label Rules (Draft)

These draft rules are based on:
- `wine-labeling-checklist.pdf`
- 27 CFR part 4
- 27 CFR part 16

Use these as implementation rules for label review workflows. This is not legal advice and does not replace full regulatory review.

## Scope

- Applies to wine labels requiring COLA review.
- Mandatory information must comply with all applicable provisions in 27 CFR part 4 and part 16.

## Rule Set

### W-01: Brand Name (Brand Label)
- Brand label must include the brand name under which the wine is sold.
- Brand name must match the COLA application brand-name entry.
- Citation: 27 CFR 4.33.

### W-02: Class/Type Designation or Statement of Composition
- Brand label must include a valid class/type designation, or a truthful and adequate statement of composition if class is not defined in regulation.
- Designation must be separate and apart from other information and must not conflict with other label text.
- If varietal names are used on brand label, ensure varietal information matches application and is approved for domestic use where required.
- If formula is required, statement of composition must match (or be more specific than) approved formula labeling instructions.
- Citation: 27 CFR 4.21, 4.34, 4.91.

### W-03: Appellation of Origin (When Triggered)
- Appellation is generally mandatory when label includes:
  - Grape varietal designation
  - Type designation of varietal significance
  - Vintage date
  - Semi-generic type designation
  - Estate bottled claim
- Appellation must appear with designation on brand label and match COLA application.
- Citation: 27 CFR 4.25, 4.34.

### W-04: Percentage of Foreign Wine (Conditional)
- For blends of American and foreign wine, if the label refers to foreign wine, it must disclose percentage composition.
- Citation: 27 CFR 4.32(a)(4).

### W-05: Alcohol Content
- Label must include alcohol content statement as required by regulation.
- Acceptable abbreviations include `Alc.`, `Alc`, `Vol.`, `Vol`, `%`.
- Citation: 27 CFR 4.36.

### W-06: Net Contents
- Net contents must appear on label or be blown/marked on container.
- Statement format must be acceptable (for example, `750 mL`, `1 L`).
- Container size must comply with approved standards of fill unless exception applies.
- Citation: 27 CFR 4.37, 4.70(b), 4.72.

### W-07: Name and Address
- Label must include responsible entity name and address (city/state) consistent with permit and application.
- DBA/trade name use must be properly reflected on permit and application.
- Name/address must immediately follow an appropriate role phrase (for example, `Bottled By`, `Imported By`) with no intervening text.
- Citation: 27 CFR 4.35.

### W-08: Sulfite Declaration
- If total sulfur dioxide is 10 ppm or greater, label must include sulfite declaration.
- If using `No sulfites added`, statement must appear with `contains naturally occurring sulfites` or `may contain naturally occurring sulfites`.
- If no sulfite declaration appears, supporting analysis may be required to substantiate less than 10 ppm sulfites.
- Citation: 27 CFR 4.32(e).

### W-09: Government Health Warning
- Label must include exact warning statement required by regulation.
- Formatting and wording checks:
  - `GOVERNMENT WARNING` in capitals and bold
  - Correct punctuation/capitalization (including Surgeon General)
  - Presented as one statement, separate and apart from other information
- Citation: 27 CFR part 16.

### W-10: Country of Origin (Imported Product Only)
- Imported wine must include country-of-origin statement in compliance with CBP requirements.
- Citation: 19 CFR 134.11.

### W-11: FD&C Yellow #5 Disclosure
- If FD&C Yellow #5 is used in wine, label must specifically disclose it.
- Citation: 27 CFR 4.32(c).

### W-12: Cochineal Extract / Carmine Disclosure
- If cochineal extract or carmine is used, label must specifically disclose it.
- Citation: 27 CFR 4.32(d).
